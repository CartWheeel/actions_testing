Weather App

Student: Jordan Lee  |  Adviser: Dr. Kim  |  Course: CS498 Senior Project  |  Semester: Spring 2025

Project Overview

Weather App is a real-time weather dashboard that fetches live data from external APIs. It is optimized for both mobile and desktop, allowing users to track conditions in multiple cities.

Problem Statement / Motivation

Many weather applications are cluttered and slow. This project aims to provide a simple, fast, and accurate weather experience.

Goals/Objectives

  • Deliver real-time updates for weather data in less than 2 seconds.
  • Ensure usability on all screen sizes.
  • Support tracking for at least 5 cities.

Key Features / Functionality

  • Live weather data using OpenWeatherMap API
  • Mobile-friendly responsive UI
  • Favorites and quick-access city list

Technologies Used

  • React
  • Node.js
  • Express
  • OpenWeatherMap API
  • CSS Flexbox/Grid

Major Challenges & Solutions

Integrating with the OpenWeatherMap API required advanced error handling for unreliable connections. Solved by implementing retry logic and graceful UI fallback.

Results / Outcomes

  • App loads city weather data in 1.5 seconds on average
  • Tested on 10+ devices with consistent results
  • Received “Best UI Design” at department expo
